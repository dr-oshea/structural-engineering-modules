Course Setup Tool

Pick a course → adjust the module set → copy the generated block into homepage/courses.js. The default set is every module from the course's prerequisite chain, computed automatically.

The courses.js key becomes e.g. CVEN2303_26T3 — use the same value in the Moodle URL's &course= parameter.

"Before You Begin" Moodle block

Generates the HTML for a Moodle Text and media area that places this course in the program: prerequisite courses (with linked refresh modules), the current course, and where it leads next. All styles are inline, so it survives Moodle's editor.

Each module links to <base><folder>/index.html.
